# Product Requirements Document: 공구마켓 (09Market) MVP

## Product Overview

**App Name:** 공구마켓 (09Market)
**Tagline:** 내가 사고 싶은 공구, 인플루언서 찾아 해매지 말고 여기서 한번에!
**Launch Goal:** 공구 정보를 찾아 헤매던 2030 여성들에게 "이거 진짜 필요했어!"라는 반응을 얻는 것
**Target Launch:** 1주일 이내 (초단기 MVP)

## Who It's For

### Primary User: 2030 공구족 김지수님
지수님은 인스타그램을 매일 사용하는 20대 후반 직장인입니다. 합리적인 소비를 좋아해서 인플루언서 공구를 자주 이용하지만, 매번 피드를 확인해야 하는 번거로움에 지쳐있습니다.

**Their Current Pain:**
- "사고 싶었던 앰플 공구, 또 놓쳤네..." (정보 휘발성)
- "이거 공구하는 사람 없나? 인스타 해시태그 뒤지기 너무 귀찮아." (정보 파편화)
- 원하는 제품이 아닌 인플루언서 위주로만 정보가 돔

**What They Need:**
- **제품 중심 검색:** "마녀공장 클렌징오일" 검색하면 공구 일정이 쭉 나오길 원함
- **한눈에 보는 일정:** 캘린더나 리스트로 시작/마감일을 직관적으로 파악하고 싶음
- **심플한 정보:** 복잡한 가입 없이 바로 정보만 딱 보고 싶음

### Example User Story
"지수님은 평소 눈여겨보던 '시카 크림'을 싸게 사고 싶습니다. 인스타그램 앱을 켜서 검색하는 대신, '공구마켓'에 접속해 '시카 크림'을 검색합니다. 이번 주에 3명의 인플루언서가 공구를 진행한다는 사실과 최저가를 확인하고, 가장 조건이 좋은 인플루언서의 링크를 타고 가서 구매합니다. 5분도 안 걸렸습니다!"

## The Problem We're Solving

인스타그램 공구 시장은 매우 크지만, 정보가 인플루언서 개개인의 피드와 스토리에 흩어져 있습니다. 소비자는 정보를 얻기 위해 수많은 계정을 팔로우하고 수시로 확인해야 하는 '노동'을 하고 있습니다. 우리는 이 노동을 없애고 **'제품'과 '가격' 중심의 합리적인 쇼핑 환경**을 제공합니다.

**Why Existing Solutions Fall Short:**
- **인스타그램 앱**: 검색 기능이 매우 약하고, 광고가 너무 많음. 정보의 휘발성이 강함.
- **네이버 카페/블로그**: 정보가 정제되지 않아 하나하나 글을 읽어야 함.

## User Journey

### Discovery → First Use → Success

1. **Discovery Phase (발견)**
   - 인스타그램 광고나 커뮤니티(더쿠, 네이트판 등) 추천글을 통해 유입
   - "여기 내가 찾던 공구 정보 다 모여있어!"라는 후기를 봄

2. **First Use (첫 사용)**
   - **첫 화면:** 군더더기 없는 심플한 검색창과 "진행 중인 인기 공구" 리스트
   - **액션:** 바로 관심 있는 브랜드나 제품명 검색 (가입/로그인 없음)

3. **Core Usage Loop (핵침 경험)**
   - 검색 결과에서 진행 중인 공구 확인
   - [구매하러 가기] 버튼 클릭 → 인플루언서 판매 페이지로 이동
   - "오, 진짜 편한데?" 느낌 (Aha Moment)

4. **Success Moment (성공)**
   - 사고 싶었던 제품을 최저가 공구로 구매 완료
   - 친구에게 "야 이거 써봐, 진짜 편해"라고 링크 공유

## MVP Features

### Must Have for Launch (P0 - 필수)

#### 1. 제품명 검색 기능
- **What:** 사용자가 브랜드명이나 제품명을 입력하면 관련 공구 정보를 찾아주는 기능
- **User Story:** "사용자는 '효소'를 검색하면 현재 진행 중이거나 예정된 효소 공구 리스트를 볼 수 있다."
- **Priority:** P0 (Critical - 이게 없으면 의미 없음)

#### 2. 공구 일정 리스트/캘린더
- **What:** 공구 시작일과 종료일(D-day)을 명확하게 보여주는 UI
- **User Story:** "사용자는 '마감 임박' 순으로 정렬하여 오늘 놓치면 안 되는 공구를 먼저 확인할 수 있다."
- **Priority:** P0 (Critical)

#### 3. 카테고리 필터
- **What:** 뷰티, 패션, 식품, 유아동 등 주요 카테고리별로 모아보기
- **User Story:** "사용자는 '뷰티' 탭을 눌러 화장품 공구만 따로 모아서 구경할 수 있다."
- **Priority:** P0 (Critical)

### Nice to Have (If Time Allows - P1)
- **[공구 알림 받기]**: 관심 키워드 등록 시 메일/카톡 알림 (Softr 폼 활용)
- **[인기 순위]**: 조회수 기반 인기 공구 랭킹

### NOT in MVP (P2 - 나중에)
- **[회원가입/로그인]**: 초기에는 진입장벽을 낮추기 위해 제외
- **[직접 결제]**: 결제 시스템 연동은 복잡하므로 아웃링크(외부 결제) 방식으로 처리
- **[커뮤니티 기능]**: 후기 공유 등은 나중에

*Why we're waiting: 1주일 내 런칭과 3만원 예산을 맞추기 위해 핵심 기능에 집중합니다.*

## How We'll Know It's Working

### Launch Success Metrics (First 30 Days)
| Metric | Target | Measure |
|--------|--------|---------|
| **일일 방문자 수 (DAU)** | 100명 | Softr/Google Analytics |
| **검색 쿼리 수** | 일 50건 | 검색창 이용 횟수 |
| **아웃링크 클릭 수** | 일 30건 | [구매하러 가기] 버튼 클릭 수 |

## Look & Feel

**Design Vibe:** 인스타그래머블, 세련된, 심플한

**Visual Principles:**
1. **이미지 중심:** 텍스트보다는 고화질 썸네일 위주로 배치 (인스타 피드 느낌)
2. **여백의 미:** 복잡한 정보는 숨기고, 핵심 정보(제품명, 가격, 날짜)만 강조
3. **트렌디한 폰트:** 가독성 좋고 세련된 산세리프 폰트 사용 (Pretendard 등)

**Key Screens/Pages:**
1. **메인 홈**: 검색창 + 카테고리 탭 + 인기 공구 배너
2. **검색 결과 페이지**: 필터링된 리스트 (카드 형태)
3. **상세 페이지**: 제품 상세 정보, 공구 일정, 인플루언서 정보, [구매] 버튼

## Technical Considerations

**Platform:** Web (모바일 웹 최적화 필수)
**Tools:** Softr (프론트엔드) + Airtable (백엔드/데이터베이스)
**Responsive:** Yes (모바일 뷰가 더 중요함)
**Budget:** 0원 ~ 2만원 (도메인 구매 비용 정도)

## Quality Standards

**What This App Will NOT Accept:**
- 모바일에서 깨져 보이는 디자인 (절대 금지!)
- 느린 로딩 속도 (이미지 최적화 필수)
- 부정확한 링크 (구매 버튼 눌렀는데 엉뚱한 곳 가면 안 됨)
- "준비 중입니다" 페이지만 가득한 메뉴

## Definition of Done for MVP

The MVP is ready to launch when:
- [ ] Airtable에 최소 30개 이상의 엄선된 공구 데이터가 입력됨
- [ ] Softr에서 검색, 필터, 리스트 조회가 완벽하게 작동함
- [ ] 모바일(스마트폰)에서 접속했을 때 디자인이 예쁘게 나옴
- [ ] 지인 3명 이상에게 테스트해보고 "쓸만하다"는 피드백을 받음
