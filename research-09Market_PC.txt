# Deep Research Report: 인스타그램 공구 통합 플랫폼 (MVP)

## 1. 요약 (Executive Summary)
- **최적의 툴 조합**: **Airtable (데이터베이스)** + **Softr (웹사이트 제작)**
  - *이유*: 무료 플랜에서 외부 사용자 접속(조회) 제한이 넉넉하고, 디자인이 깔끔하며, 검색/필터 기능이 강력합니다. (Glide는 무료 버전 업데이트 제한이 있어 공개용 서비스에는 위험할 수 있음)
- **데이터 확보 전략**: 초기 1주일은 **수동 입력(큐레이션)**이 가장 빠르고 안전합니다. 자동화 툴은 설정이 복잡하고 계정 차단 위험이 있습니다. "엄선된 정보"라는 가치를 내세우세요.
- **예산**: 0원 (도메인 구매 시 약 15,000원 ~ 20,000원/년 별도)
- **출시 목표**: 7일 완성 로드맵 포함.

## 2. 구현 도구 분석 (Tools Analysis)

### 🏆 추천: Softr + Airtable 조합
가장 많은 "정보 큐레이션" 사이트들이 사용하는 검증된 조합입니다.

| 항목 | Softr (추천) | Glide |
| :--- | :--- | :--- |
| **특징** | 레고 블록처럼 쌓아서 웹사이트 제작 | 엑셀을 앱처럼 바꿔줌 (모바일 친화적) |
| **장점** | 디자인이 예쁨, 검색/필터 기능 강력함, PC/모바일 모두 대응 | 정말 만들기 쉬움, 앱 같은 느낌 |
| **단점 (무료)** | 커스텀 도메인 불가 (softr.app 주소 사용) | **데이터 업데이트 제한(0건)**이 있어 공개 서비스 부적합 가능성 |
| **비용** | **무료** (데이터 1,000개까지) | **무료** (단, 기능 제한 큼) |
| **결론** | **공구 정보를 보여주는 '웹사이트'에는 Softr가 압도적으로 유리합니다.** | |

> **Softr 무료 플랜 특징 (2026 기준)**
> - 데이터 1,000개까지 무료 (초기 MVP로 충분)
> - 방문자 수 제한이 넉넉함
> - 검색바, 카테고리 필터(태그) 기본 제공

## 3. 데이터 확보 가이드 (Data Strategy)

### 1단계: "손품"이 명품이다 (가장 현실적인 방법)
자동 크롤링 툴(Octoparse 등)은 배우는 데 3~4일이 걸리고 인스타그램 보안 정책상 막힐 확률이 99%입니다.
1주일 내 출시를 위해 **직접 엄선한 top 50개 공구**로 시작하세요.

**[데이터베이스(Airtable) 필수 항목]**
엑셀(Airtable)에 아래 열(Column)을 만드세요.
1. **제품명** (검색용) - 예: "마녀공장 클렌징 오일"
2. **카테고리** (필터용) - 예: "뷰티", "식품", "유아동"
3. **인플루언서** - 예: "@my_daily_gonggu"
4. **공구 일정** - 시작일(Start date), 종료일(End date)
5. **구매 링크** - 프로필 링크 등
6. **썸네일 이미지** - 제품 대표 이미지 (인스타 캡처)
7. **상태** - 진행중/마감/오픈예정 (수식으로 자동화 가능)

### 2단계: 크롬 확장프로그램 활용 (보조)
- **Instant Data Scraper**: 크롬 브라우저에서 무료로 쓸 수 있는 확장 프로그램입니다. 인스타그램 PC 버전에서 리스트를 긁어올 때 시도해볼 수 있으나, 인스타그램은 구조가 복잡해 잘 안 될 수도 있습니다. (추천하지 않음)
